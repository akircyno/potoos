# Potoos Brand Planning Folder

This folder is the planning and handoff package for the Potoos rename and Poto mascot direction.

Status: **planning only**. No app rename, asset swap, OAuth change, Supabase rename, or UI redesign should happen until the mascot reference and brand direction are approved.

## Final Direction

Product: **Potoos**

Mascot: **Poto**

Tagline: **Original memories, safely shared.**

Mascot role: **Your memory guardian.**

## Files

- `brand-identity-handoff.md`
  - Product name, mascot role, palette, typography direction, voice, and implementation notes.
- `mascot-reference-prompts.md`
  - ChatGPT prompt pack for creating Poto reference images and expression sheets.
- `short-video-animation-plan.md`
  - Storyboard and tool workflow for the first mascot promo animation.
- `capcut-production-role.md`
  - Exact role of CapCut in the production workflow.
- `sprite-sheet-plan.md`
  - Beginner-friendly plan for turning Poto into usable app/game-style sprite assets.

## Handoff Use

Use this folder for:

- ChatGPT mascot reference prompts
- Claude implementation handoff later
- short video production planning
- CapCut editing workflow
- future sprite sheet planning

When the final Poto image is ready, add it to a future assets folder and update these docs with the exact filename/path.

## Important Rule

Do not rename the app in code yet. Finish mascot references and brand direction first, then do the rename in one clean implementation pass.
