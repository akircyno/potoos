# Poto Mascot Reference Prompt Pack

Use these prompts in ChatGPT Images or another image generation tool after uploading the final Poto look as reference.

## Prompting Rules

Always include:

- Product name: Potoos
- Mascot name: Poto
- Role: memory guardian
- Brand colors: deep maroon, velvet maroon, soft gold, warm cream
- Visual anchors: big golden eyes, soft feathers, camera-frame corners, gold link charm, private album/memory theme
- Style: premium cute mascot, app icon quality, polished 3D/illustrated hybrid

Best practice:

- Upload the final Poto image first.
- Tell ChatGPT to preserve the exact mascot identity.
- Generate reference sheets before animation prompts.
- Reject outputs where the eye shape, feather markings, face shape, or camera-frame identity changes too much.

Avoid:

- scary owl
- realistic predator look
- baby toy look
- messy feathers
- aggressive talons
- random social media symbols
- public likes/comments/followers
- low-quality sticker shading
