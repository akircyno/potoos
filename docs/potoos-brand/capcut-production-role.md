# CapCut Production Role

CapCut is not the tool for creating Poto's identity from scratch.

CapCut is the final editing and production control tool.

Think of CapCut as the place where the raw AI outputs become a finished brand video.

## What CapCut Should Do

Use CapCut for:

- arranging generated video clips
- trimming timing
- controlling pacing on the timeline
- keyframing simple movement such as scale, position, opacity, and gentle zooms
- adding captions
- adding the Potoos wordmark
- adding tagline text
- adding music or soft sound effects
- adding fade-ins and fade-outs
- adding light overlays and sparkles if needed
- color matching clips to maroon/gold
- creating multiple versions for different platforms
- exporting final versions

## What CapCut Should Not Do

Do not use CapCut as the main tool for:

- designing Poto's final character
- making the sprite sheet
- fixing inconsistent mascot anatomy
- creating clean transparent PNG frames
- building app-ready animation logic

If Poto's face changes too much in an AI-generated video, do not try to hide it only with CapCut. Regenerate the clip or replace it with controlled still-image animation.

## Recommended CapCut Workflow

1. Import the generated Poto video clip.
2. Import the clean Potoos logo or wordmark.
3. Import still assets:
   - final Poto PNG
   - Potoos wordmark
   - maroon/gold background
   - gold camera-frame overlay
   - sparkle overlay if needed
4. Set project format:
   - 9:16 for Reels/TikTok/Shorts
   - 1:1 for app preview/social card
   - 16:9 for website hero
5. Cut the video to 8 to 12 seconds.
6. Add text manually:
   - `Meet Poto`
   - `Your memory guardian`
   - `Potoos`
   - `Original memories, safely shared.`
7. Add gentle background music.
8. Add subtle whoosh/sparkle sounds only if they feel premium.
9. Color adjust:
   - deepen maroon shadows
   - warm gold highlights
   - avoid oversaturation
10. Export:
   - 1080x1920 MP4 for vertical
   - 1080x1080 MP4 for square
   - optional 1920x1080 MP4 for website

## CapCut's Exact Job In This Project

CapCut owns:

- final timeline
- final timing
- final text
- final music and sound
- final transitions
- final crop/aspect ratio
- final export

CapCut does not own:

- Poto's canonical design
- sprite frame cleanup
- app implementation
- brand strategy
- OAuth, Supabase, or Google Drive setup

## CapCut Text Style

Use:

- warm cream text
- soft gold accent line
- clean sans-serif
- medium or semibold weight
- no heavy outlines
- no meme-style captions

Suggested text colors:

```text
Text: #FFF8E8
Accent: #F1C85B
Shadow: #21070D at low opacity
```

## Export Naming

Use simple filenames:

```text
potoos-poto-intro-vertical-v1.mp4
potoos-poto-intro-square-v1.mp4
potoos-poto-intro-wide-v1.mp4
```

## Quality Checklist

Before export, confirm:

- Poto looks consistent in every shot.
- Text is readable on mobile.
- No weird AI artifacts are visible.
- The app promise is clear.
- The video feels premium, not childish.
- Final frame shows Potoos and the tagline.
