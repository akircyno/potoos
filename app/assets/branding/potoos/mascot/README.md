# Poto Mascot Assets

- `poto-app-icon-reference.png` is the approved app-icon style reference.

Future exports should use clear state names, for example:

- `poto-empty-album.png`
- `poto-upload-guardian.png`
- `poto-original-quality-confirmed.png`
- `poto-save-all-helper.png`
- `poto-recoverable-error.png`

Prefer transparent PNG, Rive, or Lottie exports for in-app state animations.
