# Potoos Brand Assets

This folder contains the app-ready Potoos brand assets.

Current approved direction:

- Product: Potoos
- Mascot: Poto
- Tagline: Original memories, safely shared.
- Mascot role: Your memory guardian.

Use these assets for the rebrand pass, app icon work, UI states, and motion references.

## Folder Map

- `logo/` - SVG logo mark and approved raster reference.
- `mascot/` - Poto mascot reference files and future transparent mascot exports.
- `states/` - Future UI-state assets such as empty, upload, success, error, invite, and Save All.
- `animations/` - Future frame sets, Lottie/Rive exports, or animation source notes.

Keep source references here, then export platform-specific icons into the iOS, Android, web, and PWA icon folders during the app-icon pass.
